Phase 0

VAIZOは参謀AI（SANBO）を中核に、WEB制作自動化・メディアマッチング・音声CS・マーケティングOSの6製品を展開する。 WEB制作自動化（Web Cloner + LP Generator）は既に社内稼働しPMFに最も近い。 SANBOは中小企業経営者・小規模高収益組織で高い適合度を示す。 全プロダクトがSANBOの「AI社員」として統合される設計であり、プロダクト間のクロスセルが戦略の鍵。

市場セグメント定義

VAIZO全プロダクトがターゲットとする7つのセグメント

S1. 中小企業経営者（社長）

従業員 5〜50名 / 年商 1億〜10億円 / 380万社
ペイン: 人手不足でプレイヤー兼任。戦略・資料・調査に時間が取れない。AI導入の入口が分からない。
SANBO Web Cloner Voice CS

S2. 中堅企業の経営幹部

従業員 50〜300名 / 部門長 / 5.3万社
ペイン: 部下の作業時間が経験則。AI時短の実感がない。マネジメント側のパラダイムシフトが必要。
SANBO マーケOS

S3. 小規模高収益組織

社長+1-2名 / 年商1〜3億円 / 200万+社
ペイン: 人を雇うと固定費増。AI社員で高収益維持したい。大企業のルールを避けたい。
SANBO LAPI

S4. 外貨獲得型日本企業

海外取引あり / 多言語業務 / 8.2万社
ペイン: 多言語事務・リサーチの人的コスト。翻訳ツールでは業務レベルが出ない。
SANBO

S5. 大企業の管理職

従業員300名以上 / DX推進部門 / 1.2万社
ペイン: 既存AIツール活用率低い。ROI可視化・現場浸透に苦戦。
SANBO マーケOS

S6. マーケティング責任者・広報担当

中小〜中堅 / マーケ部門 / 広報担当
ペイン: SNS運用・広告・PR・WEBサイト管理が属人的。横断的ROI把握ができない。外部委託コストが高い。
マーケOS GYAKUAPO Web Cloner LAPI

S7. 店舗・クリニック経営者

歯科・美容・飲食等 / 1〜10店舗
ペイン: 電話対応に人手を割かれる。WEB集客したいが制作費が高い。PRの仕方が分からない。
Voice CS Web Cloner LAPI GYAKUAPO

クロスセル・マッピング

1顧客に対して複数プロダクトを提案する組み合わせ戦略

🏢 S1 中小企業経営者 パッケージ

  • Entry: Web Cloner でサイトリニューアル（受託・単発）→ 信頼構築
  • Core: SANBO 5万円プランで日常業務AI化 → 体験価値
  • Expand: Voice CS で電話対応自動化 → 月額追加
  • Upsell: SANBO 30万円プラン（モバイル+活用レポート）

📈 S6 マーケ担当 パッケージ

  • Entry: LAPI でLP制作（受託）→ 速さ・品質で信頼獲得
  • Core: 統合マーケOS でSNS+広告運用 → 月額契約
  • Expand: GYAKUAPO でPR・メディア露出 → 広報拡張
  • Upsell: SANBO でマーケ部門全体のAI統括 → 経営層導入

🩺 S7 店舗・クリニック パッケージ

  • Entry: Voice CS で電話対応自動化（歯科初手）→ 即効性
  • Core: Web Cloner でホームページ刷新 → 集客基盤
  • Expand: LAPI で集客LP + GYAKUAPO でメディア露出
  • Upsell: SANBO で複数店舗の統括AI管理

🌏 S4 外貨獲得企業 パッケージ

  • Entry: SANBO で多言語リサーチ・事務処理AI化
  • Core: 統合マーケOS で海外向けSNS+広告運用
  • Expand: Web Cloner で多言語サイト自動構築
  • Upsell: SANBO 30万円 + Voice CS 多言語電話対応
