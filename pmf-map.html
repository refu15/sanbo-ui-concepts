SANBOは「少人数高権限ユーザー向けAI参謀」として、中小企業経営者と小規模高収益組織の2セグメントでPMFに最も近い。 中堅企業の部長陣・外貨獲得企業は検証フェーズ、大企業管理職は未着手。 まず経営者セグメントで確固たるPMFを確立し、商談実績をもとに拡張する戦略が最適。

市場セグメント定義

SANBOがターゲットとする5つの市場セグメント

S1. 中小企業経営者（社長）

従業員 5〜50名 / 年商 1億〜10億円
主要ペイン: 人手不足で経営者自身がプレイヤー兼任。戦略立案・資料作成・市場調査に時間が取れない。AI導入したいが何から始めるか分からない。
380万社（国内）
5万円月額単価
高導入意欲

S2. 中堅企業の経営幹部（部長陣）

従業員 50〜300名 / 部門長レベル
主要ペイン: 部下の作業時間見積もりが経験則ベース。AIで時短できるはずだが、自分がまず使いこなせていない。マネジメント側のパラダイムシフトが必要。
5.3万社（国内）
30万円月額単価
中導入意欲

S3. 小規模高収益組織

社長 + 社員1-2名 / 年商 1億〜3億円
主要ペイン: 人を雇うと固定費が増えるが、業務が回らない。AI社員で人件費を抑えながら高収益を維持したい。大企業の非本質的ルールを避けたい。
200万+社（推定）
5万円月額単価
最高導入意欲

S4. 外貨獲得型日本企業

海外取引あり / 多言語業務が発生
主要ペイン: 英語・他言語での事務・リサーチ・コミュニケーションに人的コストがかかる。翻訳ツールでは業務レベルの品質が出ない。
8.2万社（海外進出企業）
30万円月額単価
中導入意欲

S5. 大企業の管理職

従業員 300名以上 / DX推進部門
主要ペイン: 既存のAIツール導入済みだが活用率が低い。部門横断のAI活用推進、ROI可視化が求められるが、現場浸透に苦戦。
1.2万社（国内）
30万円+月額単価
低導入意欲

価値提案マッピング

SANBOのコア機能がどのセグメントの課題を解決するか

🎯 AI参謀 階層型アーキテクチャ

  • S1 / S3: 社長が方向性設定 → AI社員群が自動実行。1人で組織運営が可能に
  • S2: 部長がコンテキスト共有 → チーム全体のAI活用を統一
  • S4: 多言語AI社員を配置 → 海外事務の自動処理

📊 フォルダ別活動分析

  • S1: 部長陣の活動量を可視化 → 次の経営判断に活用
  • S2: 自部門のAI活用実績をレポート → ROI証明
  • S5: 全社AI活用率ダッシュボード（将来機能）

🔒 セキュリティ5段階モデル

  • S2 / S5: テナント分離・RLS・承認フロー → エンタープライズ要件対応
  • S4: 機密情報マスク → 海外取引データの安全な処理
  • S1 / S3: 最小構成で十分 → コスト抑制

📱 モバイル対応（上位プラン）

  • S1: PC体験後「出先でもやりたい」→ 5万円→25万円アップセル
  • S3: 少人数組織は移動が多い → モバイル必須 → 上位プラン直行
  • S2: 現場巡回中の報告・指示 → 業務効率化の実感

📝 自動フォーマット判別

  • S1 / S3: 「スライド作って」→ 自動でHTML出力。技術知識不要
  • S2: 見積書→PDF、報告書→Slides と文脈で自動判別
  • S4: 海外向け資料も同じUXで多言語生成

📈 活用レポート・提案

  • S1: 全社のAI活用状況 → コンサル的価値（同業他社展開も）
  • S2: 活用不足時に「さらに活用するには」自動提案
  • S3: 個人の活用パターン分析 → 業務自動化アイデア提案
